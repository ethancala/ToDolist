About To-Do List App

This simple to-do list application is crafted to streamline your task management. Built with HTML, CSS, jQuery, Bootstrap, and Local Storage, it offers an efficient solution for organizing your tasks effectively.

The app allows you to:

  • Add new tasks
  • Delete tasks
  • Clear all tasks
  • Edit existing tasks
  • View both active and completed tasks
